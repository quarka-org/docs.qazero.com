集客

「集客」では、ユーザーがどのような経路でウェブサイトに流入したかを詳しく分析できます。
画面上部の期間指定エリアから、レポートの対象期間を自由に変更できます。

チャネル​

サイトへの集客状況をチャネル別にレポートします。

チャネルの種類​

チャネルは、流入経路をマーケティングの視点で分かりやすくグループ化したものです。
QA ZEROでは主に以下の分類でレポートされます。

チャネル名内容
Organic Search検索エンジン（Google/Yahoo!など）の無料検索結果からの流入
Paid Searchリスティング広告などの有料検索広告からの流入
SocialSNS（Facebook, X, Instagramなど）からの流入
Emailメールマガジンや配信メール内のリンクからの流入
Affiliatesアフィリエイトサイト（提携サイト）からの流入
Displayバナー広告やディスプレイ広告からの流入
Other Advertising上記以外の広告（特定の計測パラメータが付与された広告など）からの流入
Referral他のサイトやブログ、掲示板などに貼られたリンクからの流入
Directブラウザのブックマーク、URLの直接入力、または参照元が特定できない流入
Other上記のどの条件にも合致しない分類不能な流入

参照元／メディア​

「参照元」と「メディア」を組み合わせた項目別に、より詳細な集客状況をレポートします。

参照元とは​

訪問者がサイトにアクセスする直前に滞在していたウェブサイトのURLを指します。
ただし、以下のような特定の条件下では「direct」というラベルが表示され、具体的な経路が明示されない場合があります。

  • ブラウザのアドレスバーにURLを直接入力してアクセスした場合
  • アクセスの参照元URLを特定できない場合

メディアとは​

流入経路の種類が表示されます。URLに付与されたパラメータ utm_medium の値を優先的に表示し、設定がない場合は特定のルールに基づいて自動分類します。

1. utm_medium がある場合​

流入経路を識別するためのパラメータ utm_medium をURLに付与している場合、設定した任意の値がメディア名として反映されます。特定の施策からの流入を把握できます。

  • 活用例：
    • メルマガ内のリンクに utm_medium=email と設定した場合、レポートには「email」と表示されます。
    • プレスリリースのリンクに utm_medium=press と設定した場合、レポートには「press」と表示されます。

2. utm_medium がない場合​

utm_medium が未設定の場合は、以下の条件に従って自動的に分類されます。

メディア名分類される条件
organic検索エンジン（Google/Yahoo!など）経由のアクセスの場合
cpcURLパラメータに gclid（Google広告）が存在する場合
facebookURLパラメータに fbclid（Meta広告）が存在する場合
twitterURLパラメータに twclid（X広告）が存在する場合
referral他のサイトやブログからのリンクで、参照元が特定できる場合
(none)参照元が direct（直接入力など）の場合

レポートのグラフ描画​

グラフに表示したい「チャネル」や「参照元／メディア」を個別に選んで描画できます。
表の左側にある「グラフ」列にチェックを入れて、「グラフに描画」ボタンをクリックしてください。
