# QA ZERO v3.X Release Notes

## 3.1.0.0 — 2026-04-16

### Improvements & Fixes

- Realtime screen redesign  
  See current activity more intuitively.
- Improved site switching  
  Switch between target sites easily from a select box.
- Assistant conversation screen redesign  
  A more comfortable conversation experience.
- Consolidated measurement-related settings into the Settings page  
  Authorized users can now configure them directly from the admin screen.
- Faster heatmap cache generation  
  Reduced wait time before display.
- Various minor bug fixes.

### Service Stability

- Enhanced anomaly detection  
  Strengthened mechanisms for early issue detection, including communication status checks.
