# Audience Report

The Audience report provides an overview of user activity and behavior on your site.

## Overview

You can check overall site performance for the selected period.  
Click the date area at the top of the screen to change the period.

The following metrics are displayed as graphs and values:

- **Total Users**
- **Sessions**
- **Pageviews (PV)**

---

## Users / Devices

You can analyze user activity and goal performance by **User Type** and **Device Category**.

**User Type**

- **New Visitor**: First-time visitors  
- **Returning Visitor**: Visitors who have been to the site before  

**Device Category**

- **dsk**: Desktop  
- **tab**: Tablet  
- **smp**: Smartphone  

---

## Session Recording (Video Playback)

A list of user activity data is displayed for the selected period.  
Click the icon in the **Playback** column to watch the session as a video.

You can review detailed behavior such as:

- Mouse movements  
- Page navigation flow  

---

## Advanced Export

You can download **raw data in TSV format** for detailed analysis.

The exported data includes **URLs with parameters**.
