# Session Recording

Session Recording allows you to replay user activity on your site as a video.  
You can observe mouse movements and interactions to understand user behavior beyond numbers.

It helps identify confusion, hesitation, and usability issues.

---

## How to Access Session Recordings

You can open session recordings from the following menus:

- **Realtime**
- **Visits**
- **Goals**

Click the play icon in the **Replay** column of each data list to open the player.

---

## Player Controls

### Playback Controls

Use the controls at the bottom of the screen to manage playback.

- **Speed**: 0.5x / 1x / 2x / 4x  
- **Seek**: Move to any point using the progress bar

---

### Session Information

Detailed session data is displayed on the right side.

- **User**
  - Anonymous ID (assigned by QA ZERO)
  - New / Returning
  - Country

- **Visit**
  - Start time
  - Referrer
  - Page views

- **Device**
  - Device type (Desktop / Mobile / Tablet)
  - OS
  - Browser

- **Viewed Pages**
  - Pages viewed in the same session
  - Playback automatically moves to the next page if multiple pages exist

---

## Privacy Protection

Session Recording does not capture user input.

The following data is automatically excluded:

- **Form inputs**  
  Personal data such as names, addresses, and phone numbers are not recorded.

- **Confirmation pages**  
  Pages displaying submitted data are not recorded and cannot be viewed.

No additional masking configuration is required.
