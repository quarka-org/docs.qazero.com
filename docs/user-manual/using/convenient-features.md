# Filters and Display Options

QA ZERO provides several tools to help you focus on the data you need, including date range selection, filtering, sorting, and data export.

## Change the date range

You can change the date range used in reports.

Click the date selector at the top of the screen and choose from:

- Last 7 Days
- Last 30 Days
- This Month
- Last Month
- Custom Range

For a custom range, select the start and end dates from the calendar.

## Filters

Use **Filter** at the top of each report to narrow data by specific conditions.

- **Goal completions**  
  If goals are configured, you can filter data by goal.

- **Advanced conditions**  
  Filter data using conditions such as referrer, medium, and other dimensions. Multiple conditions can be combined.

- **Sorting**  
  Click a column header to switch between ascending and descending order.

## Customize table columns

You can choose which columns are displayed in report tables.

Hide unnecessary columns and keep only the metrics you use most often to make the data easier to read.

## Chart display in Acquisition reports

In **Acquisition** reports, you can select channels or source/medium combinations to display on the chart.

Select the desired rows and click **Display on Chart**.

## Export data

Report data can be exported for further analysis.

Click **CSV Export** or **JSON Export** above the table.

## Heatmap display options

The **Heatmap** screen supports date range selection and filters like other reports.

Additional options are available:

- **Device selection**  
  Switch between Desktop, Mobile, and Tablet data.

- **Update Heatmap Version**  
  Refresh the stored page version after significant page changes.

- **Heatmap visibility**  
  Show or hide individual map types:
  - Scroll Map
  - Attention Map
  - Click Heatmap
  - Click Count Map
