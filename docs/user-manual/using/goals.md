# Goals (Conversions)

In QA ZERO, important actions such as purchases or inquiries can be defined as **Goals**.  
You can analyze the behavior of users who completed these actions.

See [How to set goals](./goal-settings.md) for setup instructions.

You can add or modify goals at any time.  
Data is recalculated based on the current settings, so you can start analysis even if goals were not configured earlier.

---

## Overview

You can view trends in goal completions and overall conversion rates (CVR).  
Use the date selector at the top to change the reporting period.

Selecting a goal filters the reports below to sessions that achieved that goal.

---

## Reports

The following reports are available:

| Report | Description |
| :----- | :---------- |
| **Source / Medium** | Identify which traffic sources contribute most to conversions |
| **Landing Pages** | Find which entry pages lead to conversions |
| **Heatmap** | Visualize where converting users focus and click |
| **Session Recording** | Watch user sessions that led to conversions |

---

## Using goal data effectively

Analyzing only numbers is not enough. Combine with other features for deeper insights:

- **Identify success patterns**  
  Find common elements viewed or clicked by converting users. Apply these patterns to other pages.

- **Find drop-off points**  
  Compare converting and non-converting users to identify where users struggle (e.g., form issues or missing information)

---

## Using goals in other reports

When goals are set, you can filter other reports by **sessions that achieved a goal**.

Simply switch the filter at the top of each report to view goal-based data.
