# Assistants

QA ZERO Assistants help you analyze your data and identify concrete actions to improve your website.  
You can interact with them in a chat-style interface without requiring specialized knowledge.

Assistants are provided as add-ons.  
You can choose and use the assistants that match your goals.  
More assistants for specific analysis needs will be added over time.

---

## Examples of Assistants

Multiple assistants are available.  
Each one focuses on a specific purpose and provides targeted insights.

Here are some examples:

### Trending Pages Analysis

Identifies pages with rapidly increasing traffic.  
Useful for spotting trends and finding ideas for new content.

### SNS Advisor

Analyzes traffic from social media such as X, Instagram, and Facebook.  
Helps you evaluate SNS performance and increase traffic.

---

## Basic Usage

1. Click **AI Assistants** in the left menu.  
2. Select the assistant you want to use.  
3. Answer the prompts (such as selecting a period), and the analysis results and suggestions will be displayed.

To switch assistants, use **-- Select AI Assistant --** at the bottom right of the screen.

---

## Working with Data (Filter & Export)

You can process and save the data displayed during the conversation.

- **Filter**: Narrow down data by keywords or values  
- **Export**: Download data in CSV or JSON format for further use

---

## Example Workflows

The conversation flow and analysis vary depending on the assistant.  
Below are examples from the initial assistants.

### Trending Pages Analysis

Identifies landing pages (LPs) that are currently growing.

Select a period:
- This week vs last week  
- Last 28 days  
- Last 90 days  

You can then view the top growing landing pages.

:::note Definition of Change

This analysis compares the **first 7 days** and the **last 7 days** within the selected period.  

For more flexible period settings, use **Rising Pages** in the **Behavior** report.

:::

Click **Analyze top pages in detail** to see traffic sources for the top pages (up to 10).  
This helps you identify the next actions to take.

---

### SNS Advisor

Analyzes traffic from social media in detail.

Select a period:
- Last 7 days  
- Last 14 days  
- Last 28 days  
- Last 90 days  

You can view traffic by referrer and medium.

Then select a specific SNS to see which landing pages users visited.  
Click **Social Media Search Result** in the table to view actual search results on SNS (not supported for all platforms).

You can switch SNS to compare results.
