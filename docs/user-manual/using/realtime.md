# Realtime

The Realtime report shows how users are interacting with your site **right now**.  
It is useful for monitoring site activity and checking immediate reactions after updates or campaigns.

---

## Realtime Overview

The current site status is displayed using two metrics.

### Users in the last 30 minutes

This shows the total number of users who performed either a **page entry** or a **page transition** within the last 30 minutes.

If a user has no activity for 30 minutes after their last action, the session is considered complete and is no longer included in this count.

### Active users in the last 1 minute

This shows the number of users who performed a **page entry** or a **page transition** within the last 1 minute.

For example, users who opened a page but did not interact (e.g., switched to another tab or remained inactive) for over a minute are not counted.

This metric represents users who are **actively viewing the site right now**.

---

## Session Recording (Video Playback)

A list of recent user sessions is displayed (up to **5,000 sessions**).

Only the most recent 5,000 sessions are available in Realtime.  
To view older sessions, use the **Audience** report, where you can select a specific period.

### Use cases

- **Check recent behavior**  
  Review sessions immediately after users leave the site. You can quickly see which pages they viewed and where they encountered issues.

- **Monitor changes and campaigns**  
  After updating content or launching a campaign, use real user behavior data to verify whether users are interacting as expected.
