# QA ZERO v3.X リリースノート

※記載している日付は更新の基準日です。各環境への適用タイミングは異なる場合があります。  

## 3.1.2.2 — 2026-05-12

### 改善・修正

- AI レポート機能を強化  
  目標一覧を AI 指示文に反映できるようになりました。  
  また、広告レポートの項目・表記を見直し、より扱いやすくなっています。

- データテーブルのデザインを刷新  
  ツールバーを整理し、列の表示／非表示を切り替えられるようになりました。  

- UTM パラメータの表示名を統一  
  分析画面と AI レポートで表記が揃い、見比べやすくなりました。

- 広告トラフィックの表示を改善  
  「参照元なし」の扱いを見直し、表示が分かりやすくなりました。

- 検索エンジン判定の改善  
  自然検索が正しく分類されないことがあった問題を修正しました。

- 広告キャンペーン別の集計を修正  
  一部の集計が正しく表示されないことがあった問題を修正しました。

- その他、翻訳・表示まわりの軽微な調整を行いました

## 3.1.0.0 — 2026-04-16

### 改善・修正

- リアルタイム画面をリニューアル  
  状況をより直感的に把握できるようになりました。
- 表示対象サイトの切り替え操作を改善  
  セレクトボックスから簡単にサイトを切り替えられるようになりました。
- アシスタント会話画面をリデザイン  
  より快適に対話できるようになりました。
- 計測関連の設定を『設定』画面に集約  
  権限を持つユーザーが、管理画面から直接設定できるようになりました。
- ヒートマップキャッシュ生成の高速化  
  表示までの待ち時間を短縮しました。
- その他、軽微な不具合を修正しました。

### サービスの安定性向上

- 異常検知機能の強化  
  通信状況のチェックなど、異常の早期検知に関する仕組みを強化しました。

## 3.0.1.0 — 2026-02-24

- ヒートマップ表示時のパフォーマンス改善
- 軽微なバグ修正

## 3.0.0.0 — 2025-11-01
