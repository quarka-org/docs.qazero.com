# QA ZERO v3.X リリースノート

## 3.1.0.0 — 2026-04-16

### 改善・修正

- リアルタイム画面をリニューアル  
  状況をより直感的に把握できるようになりました。
- 表示対象サイトの切り替え操作を改善  
  セレクトボックスから簡単にサイトを切り替えられるようになりました。
- アシスタント会話画面をリデザイン  
  より快適に対話できるようになりました。
- 計測関連の設定を『設定』画面に集約  
  権限を持つユーザーが、管理画面から直接設定できるようになりました。
- ヒートマップキャッシュ生成の高速化  
  表示までの待ち時間を短縮しました。
- その他、軽微な不具合を修正しました。

### サービスの安定性向上

- 異常検知機能の強化  
  通信状況のチェックなど、異常の早期検知に関する仕組みを強化しました。

## 3.0.1.0 — 2026-02-24

- ヒートマップ表示時のパフォーマンス改善
- 軽微なバグ修正

## 3.0.0.0 — 2025-11-01

- REST API、MCP に対応
