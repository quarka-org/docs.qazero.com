# 拡張

このセクションは、QA ZERO の上に **自分で作りたい** 人 — 自作の
プラグイン、コネクタ、連携 — と、ゆくゆくはそれを配布したい人の
ためのものです。（公式コネクタを使いたいだけなら
[連携](../integrations/index.md)を参照してください。）

:::note 準備中
作成と配布のガイドはまだ執筆中です。このページは入口を安定させる
ためのプレースホルダーで、ハウツーの内容はここに埋めていきます。
:::

## あなたが土台にするもの

あなたが作るものはすべて、他のすべてが使うのと同じ公開 REST API を
通じて QA ZERO と話します — 裏口はありません:

- **[API リファレンス](../api/index.md)** — ディスカバリの `/guide`、
  実行の `/query`。まずはここから。
- **[マテリアル](../api/materials/index.md)** — あなたの拡張が読める、
  バージョン非依存のデータ面カタログ。
- **[AI 向け — インストラクションと仕様](../for-ai/index.md)** —
  拡張がプログラム的に、あるいは LLM 経由で QAL を組み立てるなら、
  機械可読仕様（`materials.yaml`）とバージョン固定の
  `qal-validation.yaml`。

API は看板の表面であって、このセクションの下にぶら下がる依存では
ありません — 直接 API に対して作り、そこへリンクで戻してください。

## ここに載る予定のもの

- API に対してカスタムプラグインやコネクタをどう構成するか。
- サードパーティコードのための認証とバージョニングの期待値。
- 作ったものをどうパッケージし配布するか。

これらのガイドは、拡張の表面が安定し次第公開します。
