# 解約について

QA ZEROの解約をご希望の場合は、以下のフォームよりご連絡ください。

[解約申請フォーム](https://qazero.com/cancellation-request/)

## 解約の流れ

1. 上記の「解約申請フォーム」よりご連絡ください。  
2. 契約期間満了日をもって契約終了となります。  
   1. 契約期間中の途中解約はできません。契約期間満了まで引き続きご利用いただけます。  
   2. 契約期間はお申し込み時の見積書・発注書等の内容となります。  
   3. 既にお支払いいただいた利用料金の返金は原則としていたしかねます。

## 解約時のデータについて（SaaSプランのみ対象）

* 解約後、当社が別途定める期間内に、システム上のデータは削除されます。  
* 削除されたデータは復元できませんので、必要なデータは解約前にエクスポートしてください。
