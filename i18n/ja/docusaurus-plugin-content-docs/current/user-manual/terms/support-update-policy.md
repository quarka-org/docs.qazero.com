# サポートおよびアップデート

## お問い合わせ窓口

ご契約者様向けのサポート窓口をご用意しています。

[サポート問い合わせフォーム](https://qazero.com/customer-support/)  
※サポート問い合わせフォームはご契約者様専用です。アクセスにはパスワードが必要です。パスワードは納品時にご案内しています。

* 対応時間：平日 10:00〜17:30  
* 土日祝日・年末年始はお休みとなります。

## サポート対象範囲

### サポートに含まれるもの

* QA ZEROの操作方法・使い方に関するご質問  
* 不具合・障害のご報告および対応

### サポートに含まれないもの（別途お見積）

* 導入支援（初期セットアップ・環境構築など）  
* カスタマイズ開発・追加機能の開発

### プラン別のサポート提供

* SaaSプラン、サブスクプラン：月額もしくは年額費用に含まれています  
* 買い切りプラン：別途サポート＆アップデート契約が必要です

## アップデート

### SaaSプランをご利用のお客様

アップデートは当社側で自動的に適用されます。お客様側での作業は不要です。

### サブスクプラン＆買い切りプランをご利用のお客様

アップデートの提供方法はご契約内容や環境に応じて個別にご案内いたします。詳しくはサポート窓口までお問い合わせください。
