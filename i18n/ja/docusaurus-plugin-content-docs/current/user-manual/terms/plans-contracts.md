# 契約およびプラン

QA ZEROの契約の仕組みについてご説明します。  
各プランの機能や価格については、[プラン一覧ページ](https://qazero.com/lineup/)をご確認ください。

## ご契約内容の確認方法

現在ご契約中のプランや契約形態は、ご契約時にお渡ししたお見積書または契約書類に記載されています。

ご不明な場合は、[契約者専用お問い合わせフォーム](https://qazero.com/customer-support/)（パスワード入力必要・納入時のヒアリングシートに記載）よりご連絡ください。

## 契約期間と更新

QA ZEROの契約期間は、**月額プラン**と**年額プラン**からお選びいただけます。

- **月額プラン**: 1ヶ月単位でのご契約  
- **年額プラン**: 1年単位でのご契約

契約期間の満了後は**自動的に更新**されます。  
契約期間満了前にお見積書が必要な場合や解約をご希望の場合は、契約期間の満了前に[契約者専用お問い合わせフォーム](https://qazero.com/customer-support/)もしくは担当へご連絡ください。

## プラン変更

ご契約中のプランの変更（アップグレード・ダウングレード）をご希望の場合は、[契約者専用お問い合わせフォーム](https://qazero.com/customer-support/)よりご連絡ください。

変更内容に応じて、ご利用料金や適用タイミングをご案内いたします。

## お支払いについて

- **お支払い方法**: 銀行振込（請求書払い）  
- **締め日**: 月末締め  
- **お支払い期限**: 翌月末払い

### 月額プランの場合

- 初月は導入後の月末締め、翌月末払いとなります  
- 初月の請求書には**導入費用**が含まれます  
- 2ヶ月目以降は、毎月月末締め翌月末払いでの請求となります

### 年額プランの場合

- 導入後の月末締め翌月末払いでの請求となります  
- 初回の請求書には**導入費用**が含まれます

請求書やお支払いに関するご質問は、[契約者専用お問い合わせフォーム](https://qazero.com/customer-support/)もしくは担当へご連絡ください。
