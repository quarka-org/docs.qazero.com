## QA ZEROについて

QA ZERO は、プライバシーを重視した独立型分析サーバーソリューションです。

WordPress プラグインとして配布されていますが、  
一般的な WordPress 機能拡張プラグインとは異なり、  
WordPress を管理インターフェースとして利用しつつ、  
分析処理は独立したエンジンとして動作します。

そのため、設計思想や運用方法が通常のプラグインとは一部異なります。
