# ヒートマップバージョンについて

## ヒートマップバージョンとは

QA ZERO では、  
サイトページのHTMLを内部的に保存しており、  
この **QA ZERO が保持しているサイトページHTML** をもとにヒートマップを描画しています。

本プロダクトでは、この **QA ZERO が保持しているサイトページHTML** を  
**「ヒートマップバージョン」** と呼びます。

---

## ページ内容が異なって見える場合について

ヒートマップは、  
最新のヒートマップバージョン（**QA ZERO が保持している最新のサイトページHTML**）をもとに描画されます。

そのため、サイトページ内容を更新した場合、  
ヒートマップ上のページ内容が、実際のサイトページ内容と一致しないことがあります。

実際のサイトページ内容と一致させるには、ヒートマップバージョンを更新します。

---

## ヒートマップバージョン更新前の確認について

ヒートマップは常に、  
**最新のヒートマップバージョンをもとに**描画されます。  
過去のヒートマップバージョンに切り替えて表示する機能は提供していません。

そのため、**ヒートマップバージョンを更新する前に**、  
現在表示しているヒートマップ画面を  
キャプチャとして保存しておくと、  
あとから変更前後の状態を確認しやすくなります。

---

## ヒートマップバージョンを更新するには

サイトページ内容を変更したとき、  
その状態をヒートマップに反映したい場合は、  
ヒートマップ画面上部の **［ヒートマップバージョンを更新］** をクリックしてください。

この操作により、

1. 各デバイス（デスクトップ／タブレット／スマートフォン）向けのページHTMLが取得され  
2. 新しいヒートマップバージョンが作成され  
3. 画面が自動的にリロードされます  

更新後は、新しいヒートマップバージョンをもとに  
ヒートマップが表示されます。

---

## 表示ずれについて

ヒートマップバージョンの更新前後をまたぐ期間を表示している場合、  
ヒートマップの表示位置が、  
実際の操作位置と一致しないように見えることがあります。
