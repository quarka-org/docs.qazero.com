# ヒートマップ

ヒートマップでは、実際のユーザー行動を可視化し、ページ上で起きている傾向を確認できます。

ヒートマップは、「行動」配下のページ別レポートに表示される「ヒートマップ」カラムから確認できます。

選択している期間や条件に基づいたヒートマップが表示されます。


## 各ヒートマップ表示の違い

表示したいヒートマップのタイプにチェックを入れてください。

### スクロールマップ

- ページの各位置が、どの程度表示されたかを示します
- スクロール量の分布を確認できます

### アテンションマップ

- ページ上で、ユーザーの注目が集まりやすい位置を示します
- 表示時間や滞在の傾向をもとに算出されます

### クリックヒートマップ

- クリックが発生した位置を可視化します
- リンク・ボタン以外のクリックも含まれます

### クリックカウントマップ

- 各要素ごとのクリック数を数値で表示します
- クリックの集中度を把握できます


## ヒートマップの色や濃淡について

ヒートマップの色や濃淡は、ページ上で行動が発生した位置や、その集中度を相対的に示しています。


## ヒートマップ利用時の注意点

- ページビュー数と、ヒートマップ描画に反映されるデータ量は、集計の基準が異なります。
- ページ構成が変更されている場合、表示される内容が異なることがあります。
