# 計測タグの設置

## このページについて

このページでは、QA ZERO の **計測タグを Web サイトに設置する方法**について説明します。

計測タグの設置は、  
**QA ZERO の計測を開始するために必ず必要な作業**です。

---

## 計測タグとは

計測タグは、QA ZERO が発行する **専用のスクリプトタグ**です。  
このタグをサイトに設置することで、アクセスやユーザー行動の計測が行われます。

- 計測タグは **計測対象のドメインごとに発行**されます
- タグの内容は **編集せず、そのまま設置**してください
- 複数ドメインを計測する場合は、**各ドメインごとに対応するタグ**を設置してください

---

## 計測タグの設置場所

計測タグは、**各ページの `<head>` 内**に設置してください。

また、以下の点に注意してください。

- 計測対象としたい **すべてのページで読み込まれること**
- 一部のページだけに設置しないこと

---

## Google Tag Manager（GTM）の利用について（推奨）

計測タグは HTML に直接設置しても動作しますが、  
**Google Tag Manager（GTM）を利用した設置を推奨**しています。

GTM を利用することで：

- タグの管理がしやすくなる
- 実行順序を安定させやすい
- 将来的な変更にも対応しやすい

すでに他のタグ管理方法を使用している場合は、  
その方法に合わせて設置していただいて問題ありません。

---

## 設置時の注意点

- タグ設置後は、**ブラウザのキャッシュをクリアしてから**動作確認を行ってください
- 計測タグが正しく読み込まれていない場合、計測は開始されません
- 設置後、管理画面の各レポート画面でデータが反映されているかを確認してください

---

## Cookie 同意ツールを利用している場合

サイトで **Cookie 同意ツール（Cookie バナー等）を利用している場合**は、  
計測タグに加えて、**Cookie同意連携タグ** の設置が必要です。

詳しい内容は、[「Cookie同意ツールとの連携（スクリプトタグの設置）」ページ](./cookie-consent-tool-setup.md) を確認してください。


---

## 設置後の確認

- 計測タグが計測対象の全ページで読み込まれていることを確認してください
