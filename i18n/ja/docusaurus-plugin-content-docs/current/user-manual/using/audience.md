# 訪問レポート

訪問レポートでは、サイトを訪問したユーザーの属性や行動を多角的に分析できます。  

## 概要

指定した期間におけるサイト全体のパフォーマンスを把握できます。  
画面上部の日付が表示されているエリアをクリックして期間を選択してください。  

概要では、以下の指標をグラフおよび数値で表示します。  

* **総ユーザー数**  
* **セッション（訪問）数**  
* **ページビュー（PV）数**

![訪問レポート1](/img/audience/audience-1.png)

---

## ユーザー／デバイス

「ユーザータイプ」および「デバイスカテゴリー」ごとに、ユーザーの訪問状況や目標達成状況を確認できます。  

**ユーザータイプ**

* **New Visitor**: 初めてサイトを訪れたユーザー  
* **Returning Visitor**: 過去に訪問歴があるリピーター  

**デバイスカテゴリー**

* **dsk**: デスクトップPCからのアクセス  
* **tab**: タブレットからのアクセス  
* **smp**: スマートフォンからのアクセス

![訪問レポート2](/img/audience/audience-2.png)

---

## セッションレコーディング（動画再生）

指定した期間のユーザーの行動データが一覧表示されます。  
「再生」カラムのアイコンをクリックすると、実際の閲覧行動を動画として再生できます。  
ユーザーがマウスをどのように動かしたか、どのページを順に閲覧したかといった詳細な挙動の確認が可能です。  

![訪問レポート3](/img/audience/audience-3.png)

---

## Advanced Export（詳細PVデータのダウンロード）

より詳細な分析を行いたい場合に、**TSV形式の生データ**をダウンロードできます。  

データには **パラメーター付きのURLも**含まれます。
