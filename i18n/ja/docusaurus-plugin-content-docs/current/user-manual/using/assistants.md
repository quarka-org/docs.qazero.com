# AIアシスタントとの対話

QA ZEROのAIアシスタントは、蓄積されたデータから具体的な改善策を提案し、分析の手間を大幅に削減するサポート機能です。  
専門的な知識がなくても、対話形式でやり取りをしながら「次に行うべき施策」をアドバイスしてくれます。

AIアシスタントはアドオン（プラグイン）形式となっており、目的に合わせて必要なアシスタントを選択して利用します。  
今後も、特定の分析ニーズに特化した多様なアシスタントが追加される予定です。

---

## AIアシスタントの例

複数のAIアシスタントが利用可能です。  
それぞれ異なる目的に応じた分析や提案を行います。

ここでは代表的なアシスタントを紹介します。  
![アシスタント例](/img/assistants/assistants-1.png)

### 「伸びているページ分析」

アクセスが急上昇しているページや、注目を集めているコンテンツを特定します。  
流行の把握や、次に作成すべきコンテンツのヒントを得たいときに最適です。

### 「SNSアドバイザー」

SNS（X、Instagram、Facebook等）からの流入に特化した分析を行います。  
SNS運用の効果を確認したいときや、サイトへの流入をさらに増やしたいときに役立ちます。

---

## 基本的な使い方

1. 左メニューの \[AIアシスタント\] をクリックします。  
2. 利用したいAIアシスタントを選択します。  
3. 問いかけ（分析期間の選択など）に応じるだけで、分析レポートやアドバイスが表示されます。  

別のAIアシスタントに変更したい場合は、画面右下の「-- AIアシスタントを選ぶ \--」からいつでも切り替えが可能です。  

---

## データの活用（絞り込み・書き出し）

対話中に表示されるデータ一覧は、必要に応じて加工や保存が可能です。  

* フィルタ機能：特定のキーワードや数値でデータを絞り込めます。  
* エクスポート：CSVまたはJSON形式でデータをダウンロードし、他の資料作成などに活用できます。

## AIアシスタントの操作例

AIアシスタントの種類によって、対話の内容や分析項目は異なります。  
ここでは初期ラインナップのAIアシスタントの操作イメージをご紹介します。

### 「伸びているページ分析」の流れ

直近で伸びているランディングページ（LP）を特定します。

「今週と先週の変化」「直近28日間の変化」「直近90日間の変化」から期間を選ぶと、伸びているランディングページを確認できます。

:::note 変化の定義について  

この分析では、選択した期間の「最初の7日間」と「最後の7日間」を比較しています。  
より詳細な期間指定が必要な場合は、「行動」レポート内の「急上昇ページ」をご活用ください。

:::

続けて、「上位ページを詳しく分析」をクリックすると、伸び率上位（最大10ページ）の流入メディアを確認でき、次に行うべき施策のヒントが得られます。

### 「SNSアドバイザー」の流れ

SNSからの流入状況を詳しく分析します。  

「直近7日間」「直近14日間」「直近28日間」「直近90日間」から期間を選ぶと、参照元／メディア別の流入状況が表示されます。

続けて、詳しく確認したいSNSを選択すると、該当のSNSからどのランディングページにユーザーが訪れたかを表示します。  
表中の「Social Media Search Result」をクリックすると、実際のSNS上での検索結果を確認できます。（※一部のSNSには対応していません）

さらに他のSNSを選択すると、選択したSNSの分析結果を確認できます。
