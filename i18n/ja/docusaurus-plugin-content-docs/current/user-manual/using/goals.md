# 目標（コンバージョン）

QA ZEROでは、サイトでの商品購入やお問い合わせといった重要なアクションを「目標」として定義し、その成果を達成したユーザーの行動を詳細に分析できます。  
目標の登録については、[目標（コンバージョン）の設定方法](./goal-settings.md)を参照してください。

なお、**目標はいつでも追加・修正が可能**です。  
設定したタイミングで過去のデータも含めて再計算されるため、設定し忘れていた場合でも後から分析を開始できます。

---

## 概要グラフ・目標別概要

目標達成数の推移やコンバージョン率（CVR）の全体像を把握できます。  
画面上部の期間指定エリアから、レポートの対象期間を指定してください。

目標別概要にあるラジオボタンを選択すると、下部に表示されるレポートが、その目標を達成したセッションに絞り込んだ内容に切り替わります。

## レポート内容

概要グラフ・目標別概要の下には、下記4つのレポートがあります。

| 参照元／メディア | 検索エンジンや広告など、どの流入経路が最も成果に貢献しているかを特定します。 |
| :---: | :---- |
| **ランディングページ** | ユーザーが最初に訪れたページのうち、どれが最終的な成果に繋がりやすいかを分析します。 |
| **ヒートマップ** | 目標を達成したユーザーが、ページ内のどの要素を注視し、どこをクリックしたかを可視化します。 |
| **セッションレコーディング** | 目標達成に至ったユーザーの実際の画面操作を動画で確認し、成功に至るまでの迷いや決定打を把握します。 |

特に**ヒートマップ**と**セッションレコーディング**において、目標達成セッションを活用することは非常に効果的です。

* **成功パターンの抽出**: コンバージョンしたユーザーが共通して見ているコンテンツや、クリックしているボタンを特定することで、他のページやユーザーへの横展開が可能になります。  
* **離脱要因の特定**: 目標達成ユーザーの動きと比較することで、惜しくも離脱してしまったユーザーがどこでつまずいているのか（例：フォームの入力エラー、情報の不足など）をより鮮明に浮き彫りにできます。

---

## 他メニューでの活用

目標を設定すると、QA ZEROの他のメニューにある各レポートで「**目標を達成したセッション**」に絞った分析ができるようになります。  
各レポートの上部にあるラジオボタンを切り替えるだけで、特定の成果に基づいたデータを確認できます。
