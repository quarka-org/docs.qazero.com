# 集客

「集客」では、ユーザーがどのような経路でウェブサイトに流入したかを詳しく分析できます。  
画面上部の期間指定エリアから、レポートの対象期間を自由に変更できます。

![集客1](/img/acquisition/acquisition-1.png)

---

## チャネル

サイトへの集客状況をチャネル別にレポートします。

### チャネルの種類

チャネルは、流入経路をマーケティングの視点で分かりやすくグループ化したものです。  
QA ZEROでは主に以下の分類でレポートされます。

| チャネル名 | 内容 |
| :---- | :---- |
| **Organic Search** | 検索エンジン（Google/Yahoo\!など）の無料検索結果からの流入 |
| **Paid Search** | リスティング広告などの有料検索広告からの流入 |
| **Social** | SNS（Facebook, X, Instagramなど）からの流入 |
| **Email** | メールマガジンや配信メール内のリンクからの流入 |
| **Affiliates** | アフィリエイトサイト（提携サイト）からの流入 |
| **Display** | バナー広告やディスプレイ広告からの流入 |
| **Other Advertising** | 上記以外の広告（特定の計測パラメータが付与された広告など）からの流入 |
| **Referral** | 他のサイトやブログ、掲示板などに貼られたリンクからの流入 |
| **Direct** | ブラウザのブックマーク、URLの直接入力、または参照元が特定できない流入 |
| **Other** | 上記のどの条件にも合致しない分類不能な流入 |

---

## 参照元／メディア

「参照元」と「メディア」を組み合わせた項目別に、より詳細な集客状況をレポートします。

![集客2](/img/acquisition/acquisition-2.png)

### 参照元とは

訪問者がサイトにアクセスする直前に滞在していた**ウェブサイトのURL**を指します。  
ただし、以下のような特定の条件下では「**direct**」というラベルが表示され、具体的な経路が明示されない場合があります。

* ブラウザのアドレスバーにURLを直接入力してアクセスした場合  
* アクセスの参照元URLを特定できない場合 

### メディアとは

**流入経路の種類**が表示されます。URLに付与されたパラメータ `utm_medium` の値を優先的に表示し、設定がない場合は特定のルールに基づいて自動分類します。

#### 1\. utm\_medium がある場合

流入経路を識別するためのパラメータ utm\_medium をURLに付与している場合、**設定した任意の値**がメディア名として反映されます。特定の施策からの流入を把握できます。

* **活用例：**  
  * メルマガ内のリンクに utm\_medium=email と設定した場合、レポートには「**email**」と表示されます。  
  * プレスリリースのリンクに utm\_medium=press と設定した場合、レポートには「**press**」と表示されます。

#### 2\. utm\_medium がない場合

utm\_medium が未設定の場合は、以下の条件に従って自動的に分類されます。

| メディア名 | 分類される条件 |
| :---- | :---- |
| **organic** | 検索エンジン（Google/Yahoo\!など）経由のアクセスの場合 |
| **cpc** | URLパラメータに gclid（Google広告）が存在する場合 |
| **facebook** | URLパラメータに fbclid（Meta広告）が存在する場合 |
| **twitter** | URLパラメータに twclid（X広告）が存在する場合 |
| **referral** | 他のサイトやブログからのリンクで、参照元が特定できる場合 |
| **(none)** | 参照元が direct（直接入力など）の場合  |

## レポートのグラフ描画

グラフに表示したい「チャネル」や「参照元／メディア」を個別に選んで描画できます。  
表の左側にある「グラフ」列にチェックを入れて、「グラフに描画」ボタンをクリックしてください。
